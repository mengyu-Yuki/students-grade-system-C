# students-grade-system-C
基于C语言，整理学生考试成绩的简单运行程序
##项目介绍
 用C语言开发的控制台版学生成绩管理系统，支持学生成绩的增删改查并以文件的形式保存
##环境配置
 1.编译软件：Dev-C++
 2.项目结构 main.cpp
           student.cpp
           student.h
##运行步骤
 *使用Dev C++打开system项目，编译运行
 *根据菜单显示进行数据的增删改查与加载保存
 *数据保存在data.txt中
 ##功能说明
  *实现学生成绩的增删改查
  *总分按照一定的顺序排序
  *最终生成的数据保存在data.txt中
